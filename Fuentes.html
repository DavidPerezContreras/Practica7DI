Importar fuentes - @font-face y font-family

Aquí hacemos uso de una fuente local del equipo cliente

Esto lo tenemos que introducir dentro de nuestro CSS como una regla más para definir la familia de fuente a usar.

font-family: Helvetica, Verdana;

Aquí hacemos uso de una fuente importada de forma remota

En el html enlazamos un nuevo estilo desde Google fonts, dentro de la etiqueta "head"

<link rel="stylesheet" href="text/css" href="http://fonts.googleapis.com/css?family=Tangerine:bold,bolditalic|Incosolata:italic|Droid+sans">

Y dentro de nuestro fiche CSS le asignamos las familias, como haríamos normalmente con las fuentes nativas

<font-family: 'Tangerine', 'Inconsolata', 'Droid Sans', serif;>

Aquí hacemos uso de una fuente importada localmente

Para ello hacemos uso de la etiqueta "@font-face" que nos permite configurar una nueva fuente. Los parametros a tener en cuenta son "font-family" en el cual le indicamos qué nombre le queremos dar a nuestra nueva familia de fuente, y src en el cual le indicaremos la ruta hacia nuestras fuentes, que pueden estar en varios formatos distintos para asegurarnos la compatibilidad con cualquier plataforma. El resto de parámetros son opcionales y permiten configurar otros aspectos de nuestra fuente como el estilo, si debe tener los bordes suavizados, etc...

@font-face {
font-family: "FixedsysExcelsior301Regular";
src: url("https://mdn.github.io/css-examples/web-fonts/VeraSeBd.ttf");
}

Font-size

La propiedad font-size permite establecer el tamaño de letra del texto. Además de todas las unidades de medida absolutas y relativas conocidas (em, ex, px, %, in, cm, mm, pt, pc) también se pueden utilizar palabras clave que hacen referencia a tamaños de letra absolutos y relativos. Los tamaños de letra absolutos se establecen mediante las palabras clave xx-small, x-small, small, medium, large, x-large, xx-large. Aunque depende de cada navegador decidir el tamaño de las palabras clave, normalmente coinciden con los tamaños de los títulos de sección h1 a h6.

font-size: xx-large

font-size: x-large

font-size: x-large

font-size: large

font-size: large

font-size: medium

font-size: medium

font-size: small

font-size: small

font-size: x-small

font-size: x-small

font-size: xx-small

font-size: xx-small

Además de los tamaños absolutos, la propiedad font-size permite establecer el tamaño del texto de forma relativa mediante las palabras clave larger y smaller. La referencia que se toma para calcular un tamaño más grande (larger) o más pequeño (smaller) es el tamaño de letra del elemento contenedor. De todas las unidades de medida absolutas y relativas disponibles, se recomienda utilizar la unidad em para establecer el tamaño del texto. El motivo es que al tratarse de una unidad de medida relativa, se adapta fácilmente sin realizar cambios a cualquier dispositivo de acceso utilizado por los usuarios. Diferentes tamaños de texto establecidos mediante unidades relativas (en, %. px) y absolutas (pt):

Font-weight

La propiedad font-weight permite establecer el grosor con el que se muestran las letras del texto. Normalmente, la propiedad font-weight se emplea para mostrar un texto en negrita (valor bold):

font-weight: bold

Texto negrita

El uso habitual de la propiedad font-weight es el de mostrar con un estilo normal un texto que originalmente se mostraba en negrita, como por ejemplo la etiqueta strong de HTML.

font-weight: normal

Texto normal

Casi ningún tipo de letra utilizado por los usuarios normales dispone de varios grosores diferentes, por lo tanto, hasta que todos los tipos de letra no incorporen varios grosores diferentes, los valores bolder y lighter carecen de utilidad práctica.
Además, las fuentes que disponen de muchos grosores diferentes utilizan nomenclatura que no es estándar y que, por lo tanto, varía de una fuente a otra.

font-weight: 200

Texto font-weight: 200

font-weight: 900

Texto font-weight: 900

Font-style

La propiedad font-style permite establecer el estilo cone el que se muestra el tipo de letra seleccionado mediante la propiedad font-family. Los tres estilos disponibles son: normal (normal), itálico, (italic) y oblicuo (oblique)

font-style: normal

Texto style normal

font-style: italic

Texto en cursiva

font-style: oblique

Texto oblicuo

Desafortunadamente, cuando un tipo de letra dispone del estilo cursiva, los navegadores muestran el estilo oblicuo con el mismo aspecto. Por este motivo, seguramente todos los caracteres del ejemplo anterior tienen el mismo aspecto cuando los ves en cualquier navegador.

Font-variant

La propiedad font-variant permite elegir la variante con la que se muestra el tipo de letra seleccionado mediante la propiedad font-family. Las únicas variantes disponibles en el estandar oficial son: normal (normal) y versalita (small-caps). La variante versalita muestra el texto con letras mayúsculas de un tamaño similar a las letras minúsculas.

font-variant: normal

Texto normal

font-variant: small-caps

Texto versalita
