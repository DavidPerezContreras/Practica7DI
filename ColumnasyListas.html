Column-count

La propiedad column-count especifica el número de columnas en las que se debe dividir un elemento.

.Columna-count {
column-count:3;
}

Al mirar su reflejo en el agua creyó ver a otro perro con un hueso más grande que el suyo, así que intentó arrebatárselo de un solo mordisco. Pero cuando abrió el hocico, el hueso que llevaba cayó al río y se lo llevó la corriente. Muy triste quedó aquel perro al darse cuenta de que había soltado algo que era real por perseguir lo que solo era un reflejo. li

La etiqueta <li> define un elemento de lista. La etiqueta <li> se usa dentro de listas ordenadas (<ol>)y listas desordenadas (<ul>).


ul

La etiqueta <ul> define una lista desordenada (con viñetas).

<ul>
<li style="list-style-type:circle"> Cafe /li>
<li style="list-style-type:disc"> Té /li>
<li style="list-style-type:square"> Leche /li>
<li style="list-style-type:none"> Colacao /li>
<li style="line-height:200%"> Nesquikt /li>
</ul>

  • Té
  • Leche
  • Colacao
  • Nesquikt

dt

La etiqueta <dt> define un término/nombre en una lista de descripción.


dd

La etiqueta <dd> se usa para describir un término/nombre en una lista de descripción.


dl

La etiqueta <dl> define una lista de descripción.

<dl> <dt>Coffee</dt> <dd>Black hot drink</dd> <dt>Milk</dt> <dd>White cold drink</dd> </dl>

Coffee
Black hot drink
Milk
White cold drink
